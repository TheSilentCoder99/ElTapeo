# Web del Bar “El Tapeo”

Este proyecto es una página web creada para el bar de un familiar.
El objetivo principal del proyecto no fue entregar algo 100% comercial, sino aprender y practicar desarrollo web con un proyecto real, aplicando conceptos de HTML, CSS y JavaScript.

## Características

- Diseño responsive, adaptado a pantallas de escritorio, tablets y móviles.
- Carrusel de imágenes para mostrar platos, bebidas y el local.
- Secciones de especialidades, galería y contacto.
- Ejemplos de animaciones simples y efectos hover para imágenes.

## Propósito del proyecto.

Este proyecto sirvió como laboratorio personal para:

- Practicar la estructura y maquetación con HTML.
- Diseñar y estilizar elementos con CSS y media queries.
- Aplicar interactividad básica con JavaScript (carrusel, botones, navegación).
- Aprender a trabajar con un proyecto real, con imágenes, contenidos y diseño visual.
